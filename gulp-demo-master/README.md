這是一個簡單的 gulp 範例，以 gulp 執行 jshint,uglify,concat<br>

**用法**<br>
下載這個專案，在指令列輸入以下指令<br>
* 將相關的 package 下載至專案中
>npm install

* 執行 default task，此範例有語法檢查與 js 壓縮功能
>gulp


完成後到專案目錄下檢查，會看到多了 ./js/all.js 檔案
